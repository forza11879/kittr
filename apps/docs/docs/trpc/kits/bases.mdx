# List All Kit Bases

Retrieve all kit bases on kittr.

# Usage

```js
const { data: bases } = trpc.useQuery([
	"kits/bases/list",
	{ include: { count: true } }
])
```

# Returns

This query returns Kit Bases objects representing root entities that a user can a kit off of.
