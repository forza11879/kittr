# Dashboard Channel

An endpoint specifically for getting a full channel for the dashboard to work.

# Usage

```js
const { data: dashboardChannel } = trpc.useQuery([
	"channels/dashboard",
	{ id: "123", urlSafeName: "JoeWo" }
])
```
