# Update Affiliate

Update a business affiliate for a channel. Affiliates are sponsors of the stream/channel/streamer/etc.

# Usage

`code`, `company`, `description` and `url` fields are all optional.

```js
const { data: channel } = trpc.useQuery([
	"channels/profile/affiliates/update",
	{
		channelProfileId: "123a",
		code: "STREAMERMAN",
		company: "Azure",
		description: "Use code at checkout!",
		url: "https:// twitch.tv/anthonyfromkittr"
	}
])
```
